## 🕰️ 개발 기간
* 23.09.08일 - 23.09.14일

### 🧑‍🤝‍🧑 맴버구성
 - 팀장  : 신우석 - Game 로직관리
 - 팀원1 : 이소이 - 똥 프리팹 & 아이템 프리팹 & 오브젝트 풀
 - 팀원2 : 안소정 - StartScene 제작
 - 팀원3 : 임현진 - UI 총괄
 - 팀원4 : 진우성 - 캐릭터

### ⚙️ 개발 환경
- `Unity`
- `C#`
- `Github`
- `VisualStudio 2022`

## 📌 주요 기능
#### 캐릭터
- 캐릭터 이동 / 충돌 / 애니메이션
- 캐릭터 선택
#### Scenes
- 버튼을 이용하여 Scenes 이동 / 선택
#### 난이도 
- Easy / Hard 난이도 선택
- 난이도 상승

#### 싱글 플레이 / 멀티 플레이
- 플레이어 수 선택 (최대 2명)
#### 아이템
- 쉴드 아이템 사용 (1회 사용)
#### 시간 / 점수
- 시간 증가
- 피한 횟수 만큼 점수 증가
#### BGM
- 아이템 획득 / 배경음악 / 버튼 선택 / 충돌
